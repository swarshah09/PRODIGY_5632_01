# Responsive Landing Page

## [AppLink](https://yumexpress.netlify.app)

Created an interactive navigation menu that changes color or style when scrolled or when hovering over a menu item. The navigation menu have a fixed position and is visible on all pages. Used HTML to structure the menu, CSS to style it, and JavaScript to add interactivity, such as changing the background color or font color of the menu when it is scrolled or when a menu item is hovered over.
